# RotaAssist - Smart Combat Assistant | Hekili Alternative for Midnight

![RotaAssist Banner](https://raw.githubusercontent.com/yourname/RotaAssist/main/docs/images/banner.png)

**The spiritual successor to rotation helpers, built natively for WoW 12.0 Midnight.**

RotaAssist is an intelligent combat assistant designed specifically for World of Warcraft: Midnight (12.0). By leveraging Blizzard's brand-new **C_AssistedCombat** API, RotaAssist provides 100% compliance with the Terms of Service while offering visual fidelity and situational awareness far beyond legacy alternatives like SACI or JAC.

---

### Key Features

- **100% TOS Compliant**: Uses only official Blizzard APIs. No memory hacks, no automation.
- **Revolutionary T-Shaped HUD**: A modern UI layout contextually grouping your rotation, cooldowns, resources, and defensives.
- **Smart APL Forecasting**: Powered by SimulationCraft-based logic, RotaAssist predicts your next 2-3 moves.
- **Intelligent Cooldown Management**: Dynamic visual alerts when your major abilities are about to become ready.
- **Defensive Advisor**: Automatic screen alerts based on health thresholds.
- **Pre-Pull Checklist**: Automatic tracking of Food, Flasks, and Runes.
- **Native Multi-Language**: English, Simplified Chinese, and Japanese.
