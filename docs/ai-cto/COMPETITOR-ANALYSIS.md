# RotaAssist — 竞品分析 (Competitor Analysis)

## HekiLight (目前 WoW 12.0 最流行的推荐插件)
**设计哲学**:
- 极简视觉侵入：默认透明背景，只有 3-5 个图标水平排列。
- 战斗外自动隐藏，或者只显示首发技能。
- 所见即所得：第一个大图标是当前推荐，后面小图标是预测。
- 视觉权重分明：主次分明，并且绝不把 CD 中的技能放进预测条里让玩家疑惑。

**我们学到的**:
- 推荐条上不要放多余的信息（相位、准确率、资源条）。高级玩家可以有选项，但默认绝对不能混在一起。
- 距离检测 (Out-of-range) 变红和触发器发光 (Proc Glow) 是决定"游戏感"（Game Feel）的标准功能，必须有。

## Hekili (已停更的前王者)
**设计哲学**:
- 功能大而全，自带极其复杂的设置面板。
- 屏幕上可以同时出现多条序列（单体、AOE、打断）。

**我们学到的 / 差距在哪**:
- RotaAssist 通过 AI 引擎结合 Blizzard 判定自动决定 AOE/单体的最优解，不需要多条序列，这是我们产品最核心的优势。
- 但我们之前的 T 型面板太复杂，反倒走回了 Hekili 早期版本的臃肿老路。

## RotationSolverReborn (FF14)
**设计哲学**:
- 以"全自动帮打"为底层逻辑，UI 只是一个 Debug 输出窗口或者次要层。

**差距在哪**:
- RotaAssist 是纯粹的辅助 UI，绝对不帮打（符合暴雪 TOS）。
- 我们的 UI 就是核心产品，UI 的视觉感知就是一切。
